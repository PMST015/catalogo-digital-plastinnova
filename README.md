# Rediseño y Catálogo Digital - Plast-Innova 🏗️

Este proyecto consistió en el rediseño integral del sitio web de Plast-Innova, transformándolo en un potente catálogo digital para una línea de más de 200 productos industriales.

### 🛠️ Desafío Técnico: Gestión de Catálogo
El objetivo principal no era la venta directa, sino la consulta técnica. Implementé una estructura que permite:
* **Navegación de Gran Escala:** Organización de más de 200 productos en categorías lógicas.
* **Sistema de Fichas Técnicas:** Integración de archivos descargables por producto, permitiendo a los clientes obtener especificaciones industriales de forma inmediata.
* **Modo Catálogo:** Configuración personalizada para deshabilitar el carrito de compras y priorizar la consulta técnica (Showcase).

### 🌎 Internacionalización y Geolocalización (Multi-idioma)
Implementé una solución avanzada de idiomas para expandir el alcance del catálogo al mercado estadounidense:
* **Detección Automática por IP:** El sitio detecta si el usuario accede desde Estados Unidos (u otros países de habla inglesa) y cambia automáticamente el idioma del catálogo a inglés.
* **Sincronización de Contenido:** Traducción completa de los más de 200 productos y sus respectivas fichas técnicas.
* **User Experience (UX):** Reducción de la fricción al mostrar el contenido directamente en el idioma nativo del cliente potencial.

### ⚙️ Tecnologías usadas:
* **WordPress / WooCommerce** (Configurado como catálogo).
* **Gestión de Base de Datos:** Importación y organización masiva de productos.
* **UI/UX:** Diseño enfocado en el sector industrial, priorizando la claridad y la facilidad de búsqueda.

### 📈 Resultados:
* Digitalización completa del inventario físico.
* Herramienta de ventas funcional para que el equipo comercial comparta fichas técnicas con un solo clic.
